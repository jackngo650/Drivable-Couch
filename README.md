# Drivable-Couch
This will house everything I design, implement, change, and finalize on my drivable couch, inspired by the drivable couch by the 'garage goblins' on Youtube 

My plan is to use the same VESC open-source motor control firmware and inspiration of their basic motorization design, but will hope implement 2 forms of control. One wireless, either from a phone app/designing a touchpad control screen, and one wired, that will be on the actual couch. 

I do also plan on implementing other features that you'd also see on a typical car, like turn signals, lights, a speaker, etc.  

I will include my block diagrams, BOM, libraries used, and code that was used to complete this project. 

Hopefully this goes well and enjoy watching the process. I plan documenting this in a video that will be uploaded to my youtube channel and other socials.
